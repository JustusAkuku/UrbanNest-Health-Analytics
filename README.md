# CAPSTONE PROJECT
Urban Health Analytics: Insurance Charges Prediction

# Project Overview

For this project, we worked as a team of Junior Data Scientists for a healthcare-focused analytics consulting management. The company works with insurance providers to help them better understand how patient characteristics influence insurance costs.
The team's responsibility was to build a linear regression model that predicts insurance charges based on patient attributes, the Medical Cost Personal Dataset was used for this project. The dataset contained information about patients namely:age, gender, BMI, number of children, smoking status, region, and insurance charges.
